Privacy Policy

Effective Date: April 20, 2026

1. Overview

Quick Web Inspector is a browser extension designed to analyze publicly available information on websites to provide insights into performance, SEO, accessibility, and frontend technologies.

2. Data Collection

This extension does not collect, store, or transmit any personal data from users. No personally identifiable information, browsing history, or user-specific data is recorded or sent to any server.

3. How the Extension Works

All analysis is performed locally within the user's browser. The extension reads and processes the content of the currently active webpage solely to generate analysis results. This data never leaves the user's device.

4. Permissions

The extension requires certain browser permissions to function correctly. These permissions are used solely to analyze webpages and provide insights. No data is collected, stored externally, or transmitted to any server.

  • activeTab: Allows the extension to access the content of the currently active tab when the user opens the extension. This is required to analyze the webpage.
  • host_permissions (<all_urls>): Enables the extension to run on any website the user visits. This is necessary so the extension can analyze different websites on demand.
  • tabs: Used to retrieve information about the current tab (such as the tab ID and URL) in order to correctly associate analysis results with the active page.
  • storage: Used to store analysis results locally within the user's browser. No data is sent to external servers.
  • webRequest: Used to inspect network request metadata (such as headers) to detect infrastructure information like CDN or hosting providers. This permission is used in a read-only manner and does not modify or block requests.

All permissions are used strictly for the functionality of the extension. The extension does not track user activity, collect personal data, or transmit any information externally.

5. Third-Party Services

This extension does not use any third-party analytics services, tracking tools, or external APIs. All functionality is implemented using built-in browser APIs provided by the Chrome platform.

6. Changes to This Policy

This privacy policy may be updated in the future if the functionality of the extension changes. Any updates will be reflected on this page.
