# Quick Web Inspector

A lightweight Chrome extension for analyzing modern websites in real time.

Quick Web Inspector helps developers understand how a website is built, how it performs, and how to improve it by turning analysis into actionable fix recommendations directly inside the browser.

---

## Features

### Full Website Analysis
Analyze any website instantly with a structured breakdown:

- Summary with prioritized insights
- Loading Performance analysis
- Interaction Performance (visual complexity, animations, effects)
- SEO evaluation
- Accessibility checks
- Rendering strategy detection (CSR, SSR, hybrid)
- CDN & hosting detection

---

### Performance Insights

Get actionable insights instead of raw metrics:

- LCP and CLS evaluation
- JavaScript and CSS impact
- Image weight and optimization hints
- Blocking resources detection

---

### Interaction Performance (Unique Feature)

Go beyond traditional tools by analyzing UI complexity:

- Animation count and type (GPU vs layout-triggering)
- Visual effects (shadows, filters, gradients)
- DOM complexity
- Visual complexity scoring

---

### Technology Detection

Detect modern web technologies with confidence scoring:

- **18+ frontend frameworks** (Angular, React, Vue, Next, Nuxt, Astro, etc.)
- **9+ libraries** (jQuery, Tailwind, Bootstrap, Angular Material, etc.)
- **6+ CMS platforms**
- Primary vs secondary technology classification

---

### Infrastructure Insights

- CDN detection
- Hosting platform identification (e.g. Vercel)
- Resource origin analysis

---

## Popup + Full Dashboard

- **Popup view** -> quick summary + top issues
- **Full dashboard** -> complete deep-dive analysis in a new tab

---

## Installation (Manual)

The extension is currently available for manual installation while still in google's review process.

### Steps:

1. Download the project as a ZIP file
2. Extract the ZIP
3. Open Chrome and go to: *chrome://extensions/*
4. Enable **Developer Mode**
5. Click **Load unpacked**
6. Select the `dist` folder

## Purpose

This extension was built to bridge the gap between:

- Technical analysis tools (like Lighthouse)
- Real-world developer decision-making

Instead of just showing metrics, Quick Web Inspector focuses on:

- Clear explanations
- Actionable insights
- Practical performance improvements

---

## Comparison to Lighthouse

Quick Web Inspector focuses on:

- Real-time in-browser analysis
- Developer-friendly explanations
- Visual complexity & interaction performance

While Lighthouse focuses on:

- Lab-based performance metrics
- Standardized scoring

Both tools complement each other.
